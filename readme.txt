== Description ==

With this plugin, you can easily post php codes in the article.

To display a code in the article, you should use:
[phpcode]your php code[/phpcode]

The plugin is very useful & flexible.

== Installation ==

1. Upload `embed-php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use tags [phpcode]your code[/phpcode] you post php in your articles.
